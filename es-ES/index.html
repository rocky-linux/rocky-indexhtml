Bienvenido a Rocky Linux

Un sistema operativo de empresa comunitaria.

Rocky Linux es una distribución Linux de clase empresarial basado en las fuentes disponibles de CentOS Project y Red Hat que se utilizan para CentOS Stream y Red Hat Enterprise Linux. Rocky Linux tiene la intención de ser compatible con el proveedor upstream. El código fuente estará en nuestro Rocky Linux Git Service.
