Bienvenido a Rocky Linux

Un sistema operativo de empresa comunitaria.

Rocky Linux es una distribución Linux de clase empresarial basado en las fuentes disponibles de Red Hat que se utiliza para Red Hat Enterprise Linux. Rocky Linux tiene la intención de ser compatible con el proveedor upstream. El código fuente estara en nuestro Git.
